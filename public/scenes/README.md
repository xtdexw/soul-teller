## 图片规格建议

- **分辨率**：1920x1080 或更高
- **格式**：JPG 或 WebP（推荐 WebP 以获得更小的文件大小）
- **文件大小**：建议每张小于 500KB
- **宽高比**：16:9

## 场景描述参考

### 迷失的魔法王国

1. **水晶森林** - 神秘的森林，树木如水晶般透明，阳光透过树冠洒下金色光点
2. **暗影山脉** - 阴暗的山脉，充满神秘和危险的气息
3. **远古神殿** - 古老的神殿遗址，充满魔法力量
4. **王室密室** - 奢华的密室，可能有重要的秘密

### 赛博都市：霓虹之下

1. **霓虹街道** - 赛博朋克风格的街道，霓虹灯闪烁
2. **地下酒吧** - 隐秘的地下酒吧，黑客聚集地
3. **企业塔楼** - 高耸的企业大楼，冷酷而威严
4. **虚拟空间** - 数字世界的虚拟空间，充满科技感

### 江湖传说：剑影心

1. **山间小径** - 崎山小路，云雾缭绕
2. **路边茶馆** - 古朴的茶馆，江湖消息聚集地
3. **竹林深处** - 幽静的竹林，适合练功或埋伏
4. **武林大会堂** - 宏大的会堂，各路英雄齐聚

## 临时使用方案

如果您暂时没有提供图片，系统会自动使用深色渐变背景作为替代。这不影响功能的正常使用。

## 更新方式

将图片放置到对应位置后，刷新浏览器页面即可看到新的背景效果。系统会自动缓存图片，首次加载可能会有轻微延迟。
